# Graphics Coursework

## Basic Controls:

| Control | Description |
| :---: | :--- |
W    | Forward
S   | Backward
A    | Left
D    | Right
C | Toggle Camera Animation
SHIFT | Move Camera Up
SPACE | Move Camera Down

## Features Implemented:
- SplatMap 
- Bump Texture
- Imported Mesh
- Scene Graph Management
- Directional Light
- Camera Rotation around scene
- Water Texture
- CubeMaps
